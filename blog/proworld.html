带你了解编程的世界

本人将带你了解编程的世界其实嘛编程真的不难所有人都可以学习只要有 正确的学习方法正确的学习资源完完全全可以学会

首先让我们来认识一下到底什么是编程?

编程也叫编写程序是让计算机根据我们的意图来完成我们需要让计算机所 完成的事情(有点绕没关系慢慢理解)

计算机你不知道的事情

其实计算机就是一个死板的机器 只会依赖人类 计算机的世界上都是把所 有数据都是看成二进制的 二进制的世界只有0和1

如何和计算机沟通

和计算机沟通就必须要使用到计算机语言即编程语言 什么是编程语言 人 和人互相沟通需要语言 人和计算机理所当然也需要语言 但是计算机的语 言和人类的语言有着千差万别的差距

计算机语言

计算机语言就是和计算机沟通的工具

计算机语言可分为三个大类 机器语言 汇编语言 高级语言

机器语言 机器语言是机器能够直接识别和读取的语言由0和1组成

汇编语言 就是一个符号标记语言是取代机器语言的难记难查错难调试的 缺点

高级语言 高级语言就是接近人类自然语言的计算机语言可以实现高度抽 象常见的高级语言分别有 c语言 c++ java python ruby vb等等

我数学英语不好从来不及格可以学习?

可以说明编程和英语数学没有半点关系 英语:里面要很多关键字只需要你记得多写多敲多思考就可以了不需要你 英语十级这些的
数学:如果你初学编程也不要这么多数学知识点的 只是一点点小学数学知 识点例如写个基础算法这些如果你要深入了解计算机内部机制就需要你高 等数学这些知识点了但是高等数学也不难 学了高等数学有可能你一辈子 都用不到 因为计算机编程应用层编写压根不需要高等数学知识点

结论:数学不好英语不好也可以学习编程
